📂 Chapter management

The whole platform is organised chapter-wise. Create chapters here, then assign materials, tests and lecture links to them below. Drag the ↕ handles to reorder; the order shown here is the order students see.

📝 Create a test — paste the whole text, let AI arrange the questions

Paste the entire test/question paper text below in one go — no need to type each question separately. An AI model (with a rule-based fallback if no AI key is configured) will arrange it into individual questions and options. It will never guess the correct answers — review each extracted question below and set the correct option yourself; that's your answer key.

Editing an existing test — saving will update it in place.
